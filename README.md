# Swiggy Sales Forecasting

## Project Description
This project predicts food delivery sales using machine learning.

## Features
- Data Cleaning
- Exploratory Data Analysis
- Random Forest Regression
- Streamlit Dashboard
- Sales Prediction

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Matplotlib

## How to Run

### Install Dependencies

pip install -r requirements.txt

### Generate Dataset

python generate_dataset.py

### Train Model

python train_model.py

### Run Streamlit App

cd app
streamlit run app.py
